( Sassy ) дерзкий CSS

Sass (short for syntactically awesome style sheets)
это прероцессорный скриптовый язык который компилируется в Cascading Style Sheets (CSS).
SassScript дополнение для математических вычислений.

Как начать использовать scss прямо сейчас?
1. в VS code ставим плагин Live scss watch
2. создаём файл .scss
3. пишем код и он автоматически конвертируется в css

Вложеность

Sass хочет сделать вашу жизнь проще.
заместо повторения техже селекторов снова и снова вы можете переписывать их внутри родительских!
Sass будет автоматически комбинировать входящие правила и внешние.
CSS BEM 

.component {

}

.component__child-element {

}
Конкатенация SCSS for BEM 

.componen
 &__child-element {

   /*  крутая БЭМ штука только для SASS */

   }

 }
немного про переменные в CSS 

 :root{
   --css-color: red; //определение
 }
 p{
     color: var(--css-color); //вызов
 }
немного про переменные в SCSS 

$scss-color: red; //определение
 
 p{
     color: $scss-color; //вызов
     $scss-color: red; //переопределение
     /*
     переопределение внутри правила изменит только переменные внутри правил. 
     используйте !global для глобольного переопределения
     */
 }
импорт как в питоне

    /*scss*/

    p{
        font:{
            family: ... ;
            size: ... ;
        }
    }

    /*css*/

    p{
        font-family: ... ;
        font-size: ... ;
    }
шаблонные селекторы (класс который существует только в рамках scss) 

%name{ 
    color: red; 
}
.red {
     @extend %name 
}
@extend позволяет сделать наследование! 

.green{ 
    color: green; 
}
.red {
     @extend .green 
}
                    

/*на самом деле это выглядит так*/
.red, 
.green {
    color: green; 
}

mixin

позволяет делать функции, но для свойств 

@mixin font($family, $size)
{
    font-family: $family;
    font-size: $size;
}

p{
    @include font("Comic Sans", "100px")
}
    

@mixin reset-list {
    margin: 0;
    padding: 0;
    list-style: none;
  }
  
  @mixin horizontal-list {
    @include reset-list;
  
    li {
      display: inline-block;
      margin: {
        left: -2px;
        right: 2em;
      }
    }
  }
  
  nav ul {
    @include horizontal-list;
  }
